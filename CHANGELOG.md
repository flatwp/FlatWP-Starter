## [0.1.0] - 2024-12-01

### Added
- Initial release of FlatWP Starter
- Next.js 15 with App Router and Server Components
- WordPress integration via WPGraphQL
- ISR with on-demand revalidation
- Client-side search with Fuse.js
- Optimized image handling
- Preview mode for draft content
- ACF Flexible Content blocks support
- TailwindCSS v4 styling
- Shadcn/ui component library
- TypeScript strict mode
- GraphQL Code Generator integration
- Blog with posts, categories, tags, authors
- Newsletter subscription functionality
- SEO optimization (sitemap, robots.txt, meta tags)
- Responsive design
- Example ACF blocks (Hero, Features, CTA, Testimonials, Pricing)

### Documentation
- Comprehensive README with deployment instructions
- WordPress setup guide
- ACF blocks setup documentation
- GraphQL usage examples
- Quick reference guides
